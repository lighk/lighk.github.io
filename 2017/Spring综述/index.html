2. Spring 怎么就简化了Java开发呢？

Spring采取了四种策略：

  • 基于POPJ 的轻量级和最小侵入性编程；
  • 通过依赖注入和面向接口编程实现松耦合；
  • 基于切面和惯例进行声明式编程；
  • 通过切面和模板减少样板式代码。
