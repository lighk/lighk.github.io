2.代参数的decorator为什么要包三层（三阶）？

简而言之，最外层负责把decorator的参数DEBUG传进来，既然decorator携带的参数已经传进来，那么剩下的中间层和里层就是一个不带参的decorator的语法糖。反之而推，要多携带参数，肯定是需要多加一阶。
